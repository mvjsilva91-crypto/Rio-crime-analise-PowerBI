# Rio Crime Analysis – Power BI Dashboard

Projeto de análise de dados sobre criminalidade no Rio de Janeiro utilizando Power BI.

## 📊 Objetivo

Analisar dados de criminalidade no Rio de Janeiro para identificar padrões, tendências e regiões com maior incidência de crimes, auxiliando na tomada de decisões estratégicas na área de segurança pública.

## 🛠️ Ferramentas utilizadas

* Power BI
* Power Query
* DAX
* Excel
* CSV

## 📂 Estrutura do projeto

data/ → arquivos de dados utilizados na análise
excel/ → tratamento e preparação dos dados
powerbi/ → arquivos `.pbix` do dashboard
docs/ → documentação do projeto
images/ → imagens do dashboard

## 📈 Análises realizadas

* Ranking de regiões com maior criminalidade
* Classificação de risco
* Percentil de ocorrências
* Evolução temporal dos crimes
* Comparação entre períodos
